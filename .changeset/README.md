# Changesets

This folder is managed by Changesets. It records unreleased, user visible changes
so the release workflow can version the package and update the changelog.

To add a changeset, run:

```
pnpm changeset
```

Pick the change level and write a short summary. The generated markdown file is
committed alongside the change it describes. Documentation for Changesets is at
https://github.com/changesets/changesets.
